GitHub Docs Tab

A Docs tab for every GitHub repository.

GitHub access token optional

Without a token, GitHub allows 60 anonymous API requests/hour (roughly one repo listing per repo per 15 minutes thanks to caching) and only public repositories. A personal access token raises this to 5,000/hour and enables private repositories. A fine-grained token with read-only Contents permission is enough. The token is stored only in this browser's extension storage and sent only to api.github.com.

Documentation folders

One folder per line, matched case-insensitively at any nesting depth (e.g. docs also matches packages/x/docs). Multi-segment patterns like website/docs match consecutive path segments. Root-level markdown files are controlled by the checkbox below.
